# Simulation de Matchs de Premier League

Ce projet simule des **matchs de football de la saison 2023-2024 de Premier League** en utilisant des données statistiques réelles des équipes et des joueurs (passes, tirs, fautes, cartons, etc.). Il permet de prédire le **score exact des matchs** ainsi que la **performance individuelle des joueurs** (buteurs, passeurs décisifs, cartons à la minute).

## Technologies
* **Python** pour la simulation et le traitement des données
* **pandas / numpy** pour la manipulation et l'analyse des données
* **scikit-learn** pour les modèles de machine learning (RandomForest, etc.)


## Méthodes et Étapes
1. **Extraction des données** : Récupération des statistiques de la saison 2023-2024.
2. **Préparation des données** : Nettoyage, transformation et création de features avancées pour les modèles.
3. **Simulation des matchs** : Boucles simulant les événements (buts, passes, cartons) à la minute près.
4. **Prédiction ML** : Modèles pour estimer le score final et la performance des joueurs.
5. **Interface utilisateur (en développement)** : Visualisation des matchs simulés et des statistiques des joueurs.

## Comment exécuter le projet
Il faut juste run le fichier main.py, cela va prendre deux équipes aléatoires.
